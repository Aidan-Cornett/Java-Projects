B_MATH
------
The follwing is a basic program using a bunch of IF and else IF statements to make a slight introduction for myself into Java math and giving me the pain managing multiple blocks of code which is usually absent in most courses including the current one I am in.

It includes short descriptions of the following math types along with little "games":
1) Integer Math
2) Floating-point Math
3) String Comparison (as of writing no game here yet)

My favorite part was writing Overflow! which is extremely simple, but was very problematic for me trying to refuse using Copilot.

TL:DR:
* if - else if - else conditional trees
* integer, floating, arithmetic
* string comparison and formatting
* while loops
* Scanner object
